# 无限世界

> 一款基于网页的传奇类修仙 MMORPG 游戏，重温经典，探索无限可能。

三职业体系（战士/法师/道士），支持实时多人在线。零框架依赖，纯原生实现，代码即文档。

## 技术栈

| 层级 | 技术 |
|------|------|
| 前端 | HTML5 + Canvas 2D + 原生 JavaScript |
| 后端 | Node.js + Express + Socket.io |
| 数据库 | SQLite (sql.js) |
| 通信 | WebSocket 实时双向通信 |

零框架依赖，纯原生实现。

## 快速开始

```bash
# 安装依赖
npm install

# 启动游戏
npm start

# 或直接运行
node server/index.js
```

浏览器访问 http://localhost:3000，输入角色名、选择职业，即刻开始冒险。

## 游戏特色

### 三职业体系

| 职业 | 定位 | 技能 |
|------|------|------|
| **战士** | 高血量·近战·强防御 | 烈火剑法、双龙斩、逐日剑法、狂暴之力 |
| **法师** | 高魔法·远程·强输出 | 火球术、冰咆哮、雷电术、流星火雨 |
| **道士** | 均衡·治疗·召唤 | 治愈术、施毒术、召唤神兽、灵魂火符 |

### 装备系统

- **10 个装备槽位**：武器、衣服、头盔、项链、手镯×2、戒指×2、腰带、鞋子
- **7 级品质**：普通 → 优秀 → 精良 → 史诗 → 传说 → 神话 → 仙器
- **+0~+12 强化**：+4 开始有失败率，+7 以上高难度
- **宝石镶嵌**：攻击/防御/生命/魔法四种宝石，按品质开放 1~4 个槽位
- **装备鉴定**：随机附加 1~3 条额外属性
- **幸运系统**：祝福油提升幸运值，目标运 9 套
- **装备突破**：仙器可无限突破（+1, +2...），每次属性×1.5
- **套装收集**：生肖套装（12 生肖×10 阶=120 件）、五行神装（金木水火土×4 件）

### 地图与副本

- **16+ 张地图**：比奇城、盟重土城、沙巴克城、僵尸洞、沃玛寺庙、封魔谷、祖玛寺庙、赤月峡谷等
- **普通副本**：矿洞、沃玛寺庙、祖玛寺庙（独立实例）
- **无尽副本**：无限层数挑战，每 10 层一个 BOSS，掉落突破材料
- **世界 BOSS**：尸王、虹魔教主、赤月恶魔、沙巴克城主（定时刷新）

### 修仙境界

炼气 → 筑基 → 金丹 → 元婴 → 化神 → 炼虚 → 合体 → 大乘 → 渡劫 → 真仙 → 金仙 → 太乙 → 大罗 → 混元 → 道祖 → 圣人 → 天道 → 混沌 → 无极（19 境）

突破后等级不重置，第 20 境开始属性指数增长（每境×2），境界无限。

### 社交系统

- 实时聊天（世界频道 + 系统频道）
- 玩家间实时交易（金币 + 物品）
- 公会系统（创建/加入/列表）
- 野外 PK（安全区保护）
- 摆摊系统

### 特色玩法

- **自动挂机**：自动寻怪、自动攻击、自动喝药
- **挖矿系统**：矿区挖矿获取矿石材料
- **成就系统**：百斩、千斩、一代宗师等成就奖励
- **每日活动**：签到、击杀任务、BOSS 猎人等

```
    ├── test_all.js
    ├── test_playwright.js
    └── ...
```

### 核心文件速览

| 文件 | 说明 |
|------|------|
| `server/game/engine.js` | 游戏引擎核心，包含职业/装备/怪物/战斗逻辑 |
| `client/js/game.js` | 客户端完整逻辑，Canvas 渲染 + 状态同步 |
| `server/index.js` | 服务端入口，Socket.io 事件路由 |
