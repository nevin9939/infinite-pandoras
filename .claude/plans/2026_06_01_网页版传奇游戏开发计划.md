# 2026_06_01_网页版传奇游戏开发计划

## 背景

用户希望开发一个网页版传奇游戏，包含完整的传奇玩法（多职业、装备系统、副本、公会、PK），支持无限升级，使用 Node.js + SQLite 作为后端。

## 技术选型

| 层级 | 技术 | 说明 |
|------|------|------|
| 前端渲染 | HTML + JavaScript + Canvas | 2D 俯视角游戏，Canvas 足够 |
| 前端 UI | 原生 HTML/CSS | 血条、背包、技能栏等 UI 面板 |
| 后端 | Node.js + Express | RESTful API + WebSocket 实时通信 |
| 数据库 | SQLite (better-sqlite3) | 轻量级，无需额外部署 |
| 通信 | WebSocket (socket.io) | 实时战斗、PK、聊天 |

## 阶段规划

由于完整传奇玩法非常庞大，采用分阶段实现，每个阶段都可以独立运行。

---

## 阶段一：基础游戏框架（可运行的最小版本）

### 目标
- 玩家可以在地图上移动
- 遇到怪物并攻击
- 击杀怪物获得经验和升级
- 无限升级系统

### 核心设计

#### 1. 地图系统
- 使用瓦片地图（Tile-based），经典传奇的 2D 俯视角
- 地图用二维数组表示，每个格子有类型（地面、墙壁、障碍物）
- 支持多张地图（比奇城、野外、矿洞等）

#### 2. 角色系统
- **基础属性**：HP、MP、攻击、防御、速度
- **等级系统**：经验值曲线增长，无上限
  - 每级所需经验 = 基础经验 × (等级 ^ 1.5)
  - 升级时 HP/MP/攻击/防御 自动成长

#### 3. 战斗系统
- 玩家点击怪物自动寻路（A* 算法）并攻击
- 怪物有简单的 AI（发现玩家靠近则追击）
- 攻击有动画和伤害数字飘字

#### 4. 升级系统（无限）
- 击杀怪物获得经验
- 升级时属性提升
- 无等级上限
- 每 10 级解锁新技能槽位

### 具体实现步骤

1. **搭建项目脚手架**
   - 创建 client/ 和 server/ 目录
   - 配置 Express + socket.io
   - 配置 SQLite 数据库

2. **实现地图渲染**
   - 用 Canvas 绘制 2D 瓦片地图
   - 实现相机跟随（玩家始终在屏幕中央）
   - 键盘 WASD / 鼠标点击移动

3. **实现角色移动和渲染**
   - 玩家角色在地图上显示
   - 移动动画（四方向行走）
   - 服务器同步位置

4. **实现怪物系统**
   - 地图上随机生成怪物
   - 怪物在原地巡逻
   - 怪物有血条显示

5. **实现战斗系统**
   - 玩家攻击怪物
   - 怪物反击玩家
   - 死亡判定
   - 伤害数字飘字效果

6. **实现经验升级系统**
   - 击杀怪物获得经验
   - 经验条 UI
   - 升级时属性提升
   - 无限升级曲线

7. **实现数据持久化**
   - 玩家数据保存到 SQLite
   - 登录/注册
   - 加载存档
